About This Project

This project uses a Convolutional Neural Network (CNN) based on the AlexNet architecture to detect malaria in blood smear images.

It classifies cells as either Parasitized or Uninfected and further identifies the malaria species if infected. The model is designed to assist medical professionals by providing rapid preliminary analysis.

Developed by a Computer Science student as part of a project under the KTU syllabus.
